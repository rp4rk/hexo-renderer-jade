# [Jade] renderer

Add support for [Jade].

## Install

Clone the following into node modules:

``` bash
git://github.com/rp4rk/hexo-renderer-jade.git
```

## Configuration

[See Jade API for More Details](http://jade-lang.com/api/)

```yml
# Jade Compilation Options
pretty: true
debug: true

```

[Jade]: http://jade-lang.com/
